# Frontend (Next.js)

Base del frontend modular de OrlyFin.

## Fase 1
- Shell visual base.
- Route groups iniciales.
- Estructura por modulos financieros.
- Configuracion compartida para evolucion incremental.

## Fase 2
- Pantalla minima de login conectada a `/api/v1/auth/login`.
- Lectura de perfil autenticado via `/api/v1/auth/me`.

## Ejecutar
```bash
npm install
npm run dev
```
