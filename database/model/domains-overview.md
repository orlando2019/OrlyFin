# Domains Overview (DB)

Tablas y relaciones se incorporaran por dominio de manera incremental:
- Seguridad: organizaciones, usuarios, roles, permisos.
- Transaccional: ingresos, gastos, deudas, pagos, cuentas, tarjetas.
- Soporte: alertas, adjuntos, import/export.
- Control: auditoria y conciliaciones.
