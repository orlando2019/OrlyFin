# Mapa de Dominios

Dominios definidos para crecimiento incremental:

- `auth_users`
- `rbac`
- `dashboard`
- `income`
- `expense`
- `debt`
- `payment`
- `budget`
- `accounts`
- `credit_cards`
- `reconciliation`
- `alerts`
- `reports`
- `audit`
- `attachments`
- `settings`
- `import_export`

Cada dominio tendrá su propia implementación por capas, con contratos y pruebas.
