# ADR-001: Monolito Modular

## Estado
Aceptado.

## Decisión
Implementar un monolito modular con límites por dominio.

## Justificación
- Menor complejidad operativa para etapa inicial.
- Mayor velocidad de entrega.
- Facilita trazabilidad y control de seguridad centralizado.

## Consecuencia
Se evita microservicios hasta que exista evidencia de necesidad real.
