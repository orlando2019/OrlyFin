# OWASP Checklist Inicial

- [ ] Validación de input en todos los endpoints.
- [ ] Manejo centralizado de errores sin filtrar internals.
- [ ] Protección XSS/CSRF en auth web.
- [ ] Gestión de secretos fuera de código.
- [ ] Logging sin datos sensibles.
- [ ] Control de permisos por módulo/acción.
- [ ] Límites de carga y tipo para adjuntos.
